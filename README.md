Clone the Repo:
- > git clone https://github.com/rezaghiasiii/MusicxMe.git
- > cd MusicxMe
- > composer install
- > npm install
- > cp .env.example .env
- > Set up .env file
- > php artisan key:generate
- > php artisan migrate
- > php artisan serve
- > npm run dev
- [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
